# Vakit Takibi for Zen

A calm, glassy Zen Browser interface inspired by the Vakit Takibi prayer-times extension.

## What it changes

- Gives the URL bar, sidebar and menus soft translucent surfaces.
- Highlights the active tab with a clear accent indicator.
- Styles toolbar controls with subtle rounded hover states.
- Supports Zen's light and dark appearances automatically.
- Offers four accent colors and an optional compact-tab mode.

## Important

This Zen Mod changes browser appearance only. Prayer-time fetching, countdowns and notifications remain part of the separate [Vakit Takibi extension](https://github.com/selimsezr/zen_salah_extension).

## Manual testing

Copy `chrome.css` into the `chrome/userChrome.css` file in your Zen profile and restart the browser. For live editing and profile-folder instructions, see the [Zen Browser CSS guide](https://docs.zen-browser.app/guides/live-editing).
